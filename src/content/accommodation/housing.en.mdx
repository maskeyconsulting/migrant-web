# Finding Housing

<ContentSection>
  <IconHeading icon="search">House Hunting Tips</IconHeading>

### Essential Checklist

<ul>
  <li>Check building condition</li>
  <li>Verify included utilities</li>
  <li>Confirm maintenance services</li>
  <li>Review security measures</li>
  <li>Inspect common areas</li>
</ul>

### Required Documents

<ul>
  <li>Qatar ID (QID)</li>
  <li>Employment contract</li>
  <li>Salary certificate</li>
  <li>Passport copy</li>
  <li>Tenancy contract</li>
</ul>

</ContentSection>

### Steps to Rent

<ul>
  <li>Property viewing</li>

<ul>

  <li>Maintenance responsibility</li>
  <li>Notice period</li>
  <li>Utility payments</li>
</ul>
